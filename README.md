# Matplotlib CSV 可視化ツール

このプロジェクトは、気温の実況値と予測値の「誤差」を記録した CSV ファイルを読み込み、Matplotlib を使用して折れ線グラフとして可視化する Python ツールです。

## プロジェクトの概要

- **一括処理**: 指定したフォルダ内のすべての CSV ファイルを自動的にグラフ化します。
- **視認性の向上**: 1 枚の画像に最大 12 系列（12 時間分など）を描画し、データが多い場合は自動的に複数の画像に分割します。
- **誤差の強調**: 平均絶対誤差が大きい系列ほど色が薄くなるように設計されており、精度の高いデータが目立つようになっています。

---

## 準備（環境構築）

Python がインストールされている環境で、以下の手順でセットアップを行ってください。

### 1. 仮想環境の作成
プロジェクトを汚さないために、仮想環境（`env`）を作成して使用することを推奨します。

```bash
# Windows
python -m venv env
```

### 2. 必要なライブラリのインストール
`requirements.txt` を使用して、グラフ描画に必要な `matplotlib` をインストールします。

```bash
.\env\Scripts\pip install -r requirements.txt
```

---

## 使い方

### 1. グラフの生成 (`makeGraphWithCsv.py`)
CSV ファイルが格納されているフォルダと、画像を保存したいフォルダを指定して実行します。

```bash
.\env\Scripts\python -B makeGraphWithCsv.py [CSVが入っているフォルダ名] [画像を出力するフォルダ名]
```

**実行例:**
```bash
.\env\Scripts\python -B makeGraphWithCsv.py csv graphs
```
`csv` フォルダ内のすべての `.csv` ファイルが読み込まれ、`graphs` フォルダに画像が保存されます。

### 2. テストデータの生成 (`generate_csv.py`)
もし手元に CSV ファイルがない場合は、このスクリプトを実行することでテスト用のランダムなデータを生成できます。

```bash
python -B generate_csv.py
```
実行すると、`csv` フォルダ内に `temperature_YYYYMMDD_HHMMSS.csv` という名前のファイルが作成されます。

---

## CSV ファイルの形式
ツールが想定している CSV の形式は以下の通りです。

- **1 列目**: `time` (日時)
- **2 列目**: `実況値`
- **3 列目以降**: `0時間10分前`, `1時間10分前`, ..., `12時間10分前` (予測誤差)

## 補足事項
- 本ツールは Windows 環境での日本語表示（MS Gothic）を想定しています。
- グラフ内の線が薄いものは、予測と実測の乖離（平均絶対誤差）が大きいことを示しています。
